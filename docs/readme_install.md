```
$ git clone https://github.com/budlabs/i3term.git
$ cd i3term
$ make
# make install
```

#### build dependencies
[GNU make], [Gawk], [bash], [GNU sed](https://www.gnu.org/software/sed/)  

#### runtime dependencies
- [i3ass]  
- [rofi] (*to use*: `i3term --palette-menu`)  
- `xterm|urxvt|xfce4-terminal|st`
